Lion

Lion은 L10N(Localization)의 글자 장난으로, 간단한 Localization 작업을 도와주는 툴입니다. 이 툴이 할 수 있는 작업은 다음과 같습니다:

  • 지정된 xml 파일에서 string을 선택해서 excel 파일로 만들어줍니다.
  • 번역이 완료된 excel 파일을 원본 xml 파일에 치환해서 넣어줍니다.

xml 파일에서 직접 문자열 작업을 할 경우 커서 이동 등이 번거로울 수 있는데, 이 작업을 조금이나마 편하게 할 수 있도록 문자열을 excel 파일로 추출해주는 도구라고 생각하시면 되겠습니다.

툴은 다음의 위치에서 다운로드 받을 수 있습니다.

  • lion.zip
    • 위 압축 파일은 다음의 3개의 파일로 구성되어 있습니다:
    • Lion.exe
    • LionLibrary.dll
    • ClosedXML.dll

본 프로젝트의 소스코드는 이 위치에서 확인하실 수 있습니다.

사용 방법

  1. 이 링크에서 파일을 내려 받은 후, 적절한 위치에 압축을 풀어둡니다.
  2. Lion.exe 파일을 관리자 권한으로 실행합니다. 이제 xml 파일과 excel 파일 등에서 마우스 오른쪽 클릭을 하여 나타나는 shell context-menu에 Execute Lion이라는 메뉴가 추가됩니다.
  3. 변환하고자 하는 xml 혹은 excel 파일을 선택한 후 Execute Lion을 실행합니다.
  4. 만약 xml 파일을 excel 파일로 변환할 경우 어떤 attribute가 string을 포함하는지 check를 해주어야 합니다. xpath를 보고 적절히 check를 해주고 OK를 누릅니다.
  5. 만약 excel 파일을 xml 파일로 변경할 경우에는 excel 파일이 위치한 directory에 원본 xml 파일이 같이 있어야 합니다. 변환된 결과는 해당 폴더의 하위 output 폴더에 담기게 됩니다.
  6. 변환이 완료된 후 작업 로그가 나타납니다. 만약 변환이 제대로 되지 않을 경우 해당 로그를 긁어서 버그 리포트를 해주시면 됩니다.

추후 개발 내역

현재는 변환 기능이 정상 동작 여부만 판단하는 버전이므로 기능이 많이 부족합니다. 다음의 기능들이 최대한 빠르게 구현될 예정입니다:

  • xml schema 저장 기능: 한 번 schema를 만들어두면 동일한 format의 xml을 변환할 때 schema를 다시 묻지 않도록 합니다.
  • 폴더 수준의 변환: 여러 xml 혹은 excel 파일이 모여있는 폴더에 대해서도 일괄 변환이 가능하도록 합니다.
  • 용어 사전 구축: 자주 변환되는 항목에 대해 사전으로 구축하고, 이를 사용하여 번역이 좀 더 편리할 수 있도록 합니다.
  • 다양한 입력 포맷 지원: xml 뿐만 아니라 txt 혹은 word 파일에 대해서도 excel로 변환이 가능하도록 합니다.
